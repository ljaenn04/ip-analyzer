# 🔍 IP Analyzer

A web tool built with Python and Flask that analyzes any IP address and returns geolocation and network information.

## Features
- Country, city and region detection
- ISP and organization identification
- Coordinates and timezone
- Clean and responsive web interface

## Tech Stack
- Python 3
- Flask
- ip-api.com API
- HTML/CSS

## How to run

```bash
git clone https://github.com/ljaenn04/ip-analyzer.git
cd ip-analyzer
pip install flask requests
python app.py
```

Then open http://127.0.0.1:5000 in your browser.
